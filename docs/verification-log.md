# 実機照合ログ（Switch版）

実機（ゲーム）とアプリの値を突き合わせ、Champions固有差分を確定するための記録様式。
判定が `差分あり` のものは `差分案` を `BUILTIN_PATCH` / 配信JSON（`docs/patch.example.json` 形式）へ反映する。

**共通条件（特記なき限り）:** Lv50・IV31・特性/道具なし・天候/フィールドなし・非急所。

---

## C. 差分案（確定したものを転記 → patch へ）

```json
{
  "version": "2026.0X-A",
  "updatedAt": "YYYY-MM-DD",
  "moves": {
    "（技名）": { "basePower": 0, "verified": true, "note": "実機照合" }
  },
  "species": {
    "（種族名）": { "baseStats": { "def": 0 }, "verified": true, "note": "実機照合" }
  }
}
```
