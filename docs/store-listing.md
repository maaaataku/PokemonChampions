## 説明文（本文）

```
ゲーム内ランク対戦（ダブル中心）の「実戦コンパニオン」。
対戦前後の高ストレスな状況で、最短タップで確定数と調整がわかるスマホ特化のダメージ計算ツールです。

■ 特長
・ダブル主軸の2v2盤面。範囲技は複数対象を同時表示、味方の巻き込みも警告
・確定数を主役に大表示。色で危険度をひと目で把握
・合算KO（2体で1体を落とせるか）を切り替えなしで判定
・耐久調整＝この攻撃を「確定耐え」する最小投資を逆算してワンタップ反映
・素早さ早見＝スカーフ・追い風・まひ・トリックルーム込みの行動順
・タイプ相性・対策＝防御相性/弱点/有効打/パーティ補完。テラスタルで変わる相性も
・全技ダメ計・与/被ダメの両方向計算
・構築の保存・読込、共有コードでの受け渡し、Showdown/PokePaste の取込・書き出し

■ 計算の正確さ
ダメージ計算は @smogon/calc（Gen9データ）をベースに、能力ポイント制
（Lv50・個体値固定）を厳密に再現しています。ダブル補正（範囲×0.75・壁・てだすけ・
いかく・フレンドガード・天候/フィールド）もエンジンが正しく計算します。

■ オフライン対応
電波の悪い大会会場でも使えるよう、計算はすべて端末内で完結します。

■ 開発を応援
無料・広告なしで提供しています。応援いただける方は、開発者ページ（GitHub Sponsors）からご支援いただけます。
※アプリ内のチップ（任意）でも応援いただけます。

■ ご注意
・本アプリは非公式のファンメイドツールです。任天堂株式会社・株式会社ポケモン等とは関係ありません。
・ゲーム固有の調整値は実機照合で順次反映します。最新の数値と差異がある場合があります。
・広告なし・トラッキングなし・個人情報の収集なし。
```
